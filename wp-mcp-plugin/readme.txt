== Description ==

Bridge the gap between AI assistants and WordPress! This plugin enables Claude (or any MCP-compatible AI) to manage your WordPress site through natural language. Built with love, tested in production, and ready to transform how you work with WordPress.

From content creation to WooCommerce management, from template editing to bulk operations - your AI assistant can now DO things, not just talk about them.

Made with ❤️ by Breuk & Claude - Intelligence Combined!

== Features ==

* Complete WordPress control via AI assistants
* WooCommerce product and order management
* Safe template editing with automatic backups
* Bulk operations that actually work
* Security-first design with rate limiting
* Application Password authentication
* Production-ready from day one

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate through the 'Plugins' menu
3. Generate an Application Password in Users → Your Profile
4. Configure your MCP server with the credentials
5. Start talking to Claude about your WordPress site!
